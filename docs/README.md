# docs/

This directory contains technical documentation for the `otel-gitops`
repository.

| File | Contents |
|------|----------|
| `architecture.md` | System architecture, GitOps workflow, and design decisions |
| `runbook.md` | Operational runbook with common commands and troubleshooting |

## Contributing to docs

- Keep documentation close to the code it describes.
- Update `runbook.md` whenever a new operational procedure is established.
- Use diagrams (Mermaid or ASCII) where possible.
